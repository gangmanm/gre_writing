# Issue Task 작성 가이드

GRE Issue Task는 주어진 주제에 대해 자신의 입장을 논리적으로 전개하는 에세이입니다. 다양한 사회적, 교육적, 문화적 주제에 대한 비판적 사고력과 논증 능력을 평가합니다.

@info[시험 개요]
Issue Task는 30분 동안 진행되며, 주어진 주제에 대해 자신의 입장을 논리적으로 전개하는 500-600단어 분량의 에세이를 작성해야 합니다.
@end
