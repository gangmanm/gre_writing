# GRE Argument Task Guide

GRE Argument Task는 주어진 논증을 분석하고 평가하는 과제입니다. 논증의 논리적 허점을 찾아내고, 이를 체계적으로 비판하는 능력을 측정합니다.

@info[시험 개요]
Argument Task는 30분 동안 진행되며, 주어진 논증의 논리적 타당성을 평가하고 비판하는 에세이를 작성해야 합니다.
@end
